Understanding the HTML Figure Tag

HTML Theory: Tags, Elements, and Attributes

HTML Tags

Tags are the fundamental building blocks of HTML. They are enclosed in angle brackets < > and usually come in pairs:

  • Opening tag: <tagname>
  • Closing tag: </tagname>
  • Example: <figure> and </figure>

HTML Elements

An HTML element consists of a start tag, content, and an end tag:

  • The <figure> element is used to mark up a self-contained content unit
  • The <figcaption> element provides a caption for the figure
  • The <img> element is a self-closing tag used to embed images

HTML Attributes

Attributes provide additional information or properties to HTML elements:

  • src: Specifies the source URL of an image
  • alt: Provides alternative text for accessibility
  • width/height: Defines dimensions of an element
  • class: Assigns one or more CSS classes
  • id: Provides a unique identifier
