Meta Tags in HTML

Meta tags are crucial HTML elements that provide metadata about your webpage. They help with SEO, social sharing, character encoding, and mobile responsiveness.

Character Encoding (charset)

The UTF-8 charset meta tag ensures proper encoding of special characters and symbols across different languages and systems:

<meta charset="UTF-8">

Viewport Settings

The viewport meta tag controls how your website appears on mobile devices:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

- width=device-width: Sets the width of the page to follow the screen-width of the device
- initial-scale=1.0: Sets the initial zoom level when the page is first loaded

SEO Meta Tags

These tags help search engines understand and rank your page:

  • Description: Provides a brief summary of the page content (155-160 characters recommended)
  • Keywords: Although less important now, these help categorize your page content
  • Author: Specifies the page's author

Open Graph Meta Tags

Open Graph (og) tags control how your page appears when shared on social media:

  • og:title: The title displayed when shared
  • og:description: The description shown in social media previews
  • og:image: The image displayed in social media shares
  • og:url: The canonical URL for the page

Cache Control

These meta tags control how browsers cache your page:

  • Cache-Control: Directs browsers not to cache the page
  • Pragma: Legacy cache control for HTTP/1.0 compatibility
  • Expires: Sets when the page cache should expire

Page Refresh

The refresh meta tag automatically reloads the page after a specified time:

<meta http-equiv="refresh" content="30">

In this example, the page refreshes every 30 seconds

Theme Color

The theme-color meta tag sets the color of the browser's UI elements on mobile devices:

<meta name="theme-color" content="#4285f4">

This affects elements like the address bar in mobile browsers

Favicon

While not a meta tag, the favicon link defines the icon shown in the browser tab:

<link rel="icon" href="favicon.ico" type="image/x-icon">
