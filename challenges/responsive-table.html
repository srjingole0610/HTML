Goal: Learn how to use HTML table elements, including semantic grouping, headers, and merging cells.

üìã Challenge Requirements:

Create an HTML page that displays a student grade report using a well-structured table.

The table should:

  1. Include the following columns:

    • Student ID
    • Name
    • Subject
    • Marks Obtained
    • Grade
    • Remarks

  2. Use proper HTML table structure:

    • <table> to define the table
    • <thead>, <tbody>, and <tfoot> to organize the sections
    • <th> for headers and <td> for data cells
    • Use at least one colspan and rowspan in a meaningful way (e.g., merge header cells or repeated student names)

  3. Add a caption:

    • Use the <caption> tag to add the title: "Student Performance Report - 2025"

  4. Use HTML attributes or CSS for styling:

    • Add border via inline style or CSS
    • Add cell padding and alternate row colors (zebra stripes)

üìå Bonus (Optional):

  • Add a footer row showing the average score or number of students
  • Add a tooltip (title attribute) on each grade column to explain what it means (e.g., "A: Excellent")
