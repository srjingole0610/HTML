HTML Form Elements Theory

Form Basics

HTML forms are used to collect user input. The <form> element defines a form that contains form elements. Forms can submit data to a server for processing.

Common Form Attributes

  • action: Specifies where to send form data when submitted
  • method: Defines HTTP method (GET/POST) for sending data
  • enctype: Specifies how form data should be encoded
  • autocomplete: Enables/disables browser autocomplete functionality

Input Types

  • text: Single-line text input
  • password: Masked text input for sensitive data
  • email: Input field for email addresses with validation
  • number: Numeric input with optional controls
  • tel: Input for telephone numbers
  • url: Input for web addresses
  • search: Search input field
  • date: Date picker input
  • time: Time picker input
  • datetime-local: Date and time picker
  • month: Month picker
  • week: Week picker
  • color: Color picker
  • range: Slider control
  • file: File upload control
  • hidden: Hidden input field

Form Controls

  • radio: Single-selection buttons
  • checkbox: Multiple-selection boxes
  • textarea: Multi-line text input
  • select: Dropdown selection menu
  • button: Clickable button elements

Common Attributes

  • required: Makes a field mandatory
  • disabled: Disables user interaction
  • readonly: Makes field read-only
  • placeholder: Provides hint text
  • pattern: Defines validation pattern
  • min/max: Sets minimum/maximum values
  • step: Defines increment steps
